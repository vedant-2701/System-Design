```text
/
├── bin/      → essential binaries (ls, cp, bash)
├── sbin/     → system binaries (for root, e.g. iptables, fdisk)
├── etc/      → configuration files
├── var/      → variable data (logs, caches, spool)
├── proc/     → virtual, live kernel/process state
├── sys/      → virtual, hardware and driver state
├── tmp/      → temporary files, cleared on reboot
├── home/     → user home directories
├── root/     → root user's home
├── usr/      → user-installed programs and libraries
├── lib/      → shared libraries for /bin and /sbin
├── dev/      → device files
├── mnt/      → mount points for filesystems
├── opt/      → optional/third-party software
├── run/      → runtime data (PIDs, sockets)
```
