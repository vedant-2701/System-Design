# starter_prompts.md

## Purpose

Reusable starter prompts for different chat types.

These prompts are intentionally:
- short
- modular
- context-efficient

The uploaded markdown files contain the real behavioral instructions.

These prompts should only:
- activate the correct files
- define the topic
- define the focus

Avoid giant prompts.

---

# Usage Philosophy

These prompts are intentionally lightweight.

The uploaded markdown files contain:

* behavioral rules
* teaching frameworks
* operational logic

These starter prompts should only:

* activate the right context
* define the task
* define the focus

Avoid creating giant mega-prompts.
